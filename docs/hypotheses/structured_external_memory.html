理论展开

这条假设关注的是长期记忆从“更大上下文窗口”转向“结构化外部对象”的范式变化。单纯扩大窗口会同时放大成本、延迟、噪声和检索困难，而 memory object、graph、session state 允许系统只读取当前任务真正需要的片段。它的意义不只是省 token，而是把记忆变成一个可维护、可选择、可审计的独立层。若该方向成立，未来长程 agent 和知识系统会更像数据库与工作内存的组合，而不是一个一次性吞下全部历史的巨型 prompt。
