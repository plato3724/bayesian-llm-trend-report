本轮结论

  1. 第一版后验更支持“agent 基础设施化”而不是“更会聊天”。Anthropic 和 Claw AI Lab 都把 session、sandbox、harness、verified-result loop 放到了核心位置。
  2. 第二个强信号不是更长上下文，而是更外部化、可查询、可结构化的 memory/object/graph。MemPalace 与 Graphify 都在绕开“把一切都塞进窗口”。
  3. 第三个稳定趋势是 token efficiency 变成独立设计轴，而不是附属优化项。Graphify、caveman、MemPalace 都把 token/cost/latency tradeoff 写进产品形态里。
  4. 语音方向不再只是 TTS。Any2Speech 与 VoxCPM2 共同指向角色、情绪、场景、长音频和 voice design 的统一建模。
  5. 软件工程趋势上，Simon Willison 的一手总结与 agent/harness 项目相互印证：实现变快之后，真正的稀缺能力转向验证、测试、编排与判断。
  6. 多模态外延继续向物理世界推进。EgoTouch/TouchAnything 代表的是 tactile grounding，而不是单纯再加一种输入模态。
